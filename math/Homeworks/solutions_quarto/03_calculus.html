Problem 1: Limits of Sequences

Does the following sequence have a limit (as \(n\to\infty\))? If so, find it.

  1. \(\dfrac{3-n}{2}\)

  2. \(\dfrac{4\sqrt{n}}{n^3}\)

  3. \(\dfrac{n-1}{n^2-1}\)

  4. \(1^n\)

  5. \(0.4^n\)

  6. \((-4)^n\)

  7. Choose any number as the first term; divide it by \(1.5\) to get the next term; then repeat the same step again and again.

  8. \(a_1=1\); \(a_2 =\) the first digit of \(a_1/7\) after the decimal point; \(a_3 =\) the first digit of \(a_2/7\) after the decimal point; \(\dots\)

Solutions:

  1. \(\dfrac{3-n}{2} = -\dfrac{n}{2}+\dfrac{3}{2} \to -\infty\). No finite limit; the sequence diverges to \(-\infty\).

  2. \(\dfrac{4\sqrt{n}}{n^3}=4n^{-5/2}\to 0\).

  3. \(\dfrac{n-1}{n^2-1}=\dfrac{n-1}{(n-1)(n+1)}=\dfrac{1}{n+1}\) (for \(n\neq 1\)), hence \(\to 0\).

  4. \(1^n=1\) for all \(n\); limit \(=1\).

  5. \(|0.4|<1\Rightarrow 0.4^n\to 0\).

  6. \((-4)^n\) has magnitude \(4^n\to\infty\) and alternates in sign; no limit (diverges).

  7. If \(a_{n+1}=\dfrac{a_n}{1.5}=\dfrac{2}{3}a_n\), then \(a_n=a_1\left(\dfrac{2}{3}\right)^{n-1}\to 0\) for any \(a_1\). Limit \(=0\).

  8. \(a_1=1\). Since \(1/7=0.\overline{142857}\), the first digit after the decimal is \(1\); thus \(a_2=1\). Repeating gives \(a_3=1\), etc. So \(a_n=1\) for all \(n\ge 1\); limit \(=1\).

Problem 2: Continuity

The following functions consist of two “parts” which are glued together at \(x=2\). Is there a value of \(c\) such that the functions are continuous at \(x=2\)?

  1. \(f(x)=\begin{cases} 3x-5,& \text{if } x< 2 \\ x^2+c,& \text{if } x\ge2 \end{cases}\)

  2. \(f(x)=\begin{cases} x^3+1,& \text{if } x<2 \\ cx^2,& \text{if } x\ge2 \end{cases}\)

  3. \(f(x)=\begin{cases} -7,& \text{if } x<2 \\ c,& \text{if } x=2 \\ 4+3\sin(\pi x),& \text{if } x>2 \end{cases}\)

  4. \(f(x)=\begin{cases} \dfrac{x^2-x-2}{x-2},& \text{if } x\neq 2 \\ c,& \text{if } x=2 \end{cases}\)

Solutions:

  1. For continuity at \(x=2\): \(\lim_{x\to 2^-} f(x) = \lim_{x\to 2^+} f(x) = f(2)\)
    • \(\lim_{x\to 2^-} (3x-5) = 3(2)-5 = 1\)
    • \(f(2) = 2^2+c = 4+c\)
    • For continuity: \(1 = 4+c \Rightarrow c = -3\)
  2. For continuity at \(x=2\):
    • \(\lim_{x\to 2^-} (x^3+1) = 8+1 = 9\)
    • \(f(2) = c(2)^2 = 4c\)
    • For continuity: \(9 = 4c \Rightarrow c = \dfrac{9}{4}\)
  3. For continuity at \(x=2\):
    • \(\lim_{x\to 2^-} (-7) = -7\)
    • \(\lim_{x\to 2^+} (4+3\sin(\pi x)) = 4+3\sin(2\pi) = 4+3(0) = 4\)
    • Since left and right limits differ (\(-7 \neq 4\)), no value of \(c\) makes the function continuous at \(x=2\).
  4. First simplify \(\dfrac{x^2-x-2}{x-2} = \dfrac{(x-2)(x+1)}{x-2} = x+1\) for \(x\neq 2\)
    • \(\lim_{x\to 2} (x+1) = 3\)
    • For continuity: \(c = 3\)

Problem 3: Investment Growth (Limits)

Zeus, Prometheus and Aramazd invest 1000 gold coins each in different banks with growth functions: - OlympusBank: \(f(x)=100x+x^3\) - TaurusBank: \(f(x)=0.18x^3\sqrt{x} = 0.18x^{7/2}\) - AraratBank: \(f(x)=30x^2\)

Who will be richest after infinitely long time?

Solution:

Compare the growth rates as \(x \to \infty\): - OlympusBank: \(100x+x^3 \sim x^3\) (cubic growth) - TaurusBank: \(0.18x^{7/2}\) (power \(7/2 = 3.5\)) - AraratBank: \(30x^2\) (quadratic growth)

Since \(7/2 > 3 > 2\), TaurusBank grows fastest. Prometheus will be the richest.

Problem 4: Derivatives

Find the derivatives of the following functions:

  1. \(f(x)=10000\)
  2. \(f(x)=2x^2-7x+1\)
  3. \(f(x)=2\sin x \cdot e^x\)
  4. \(f(x)=e^{x^2}\)
  5. \(f(x)=x^2\ln x\)
  6. \(f(x)=\dfrac{x^2}{x^3}\)
  7. ReLU: \(f(x) = \begin{cases} x, & \text{if } x>0\\ 0, & \text{if } x\le 0 \end{cases}\)
  8. Sigmoid: \(f(x) = \dfrac{1}{1+e^{-x}}\)

Solutions:

  1. \(f'(x) = 0\) (derivative of constant)

  2. \(f'(x) = 4x - 7\) (power rule)

  3. \(f'(x) = 2(\sin x \cdot e^x)' = 2(e^x \sin x + e^x \cos x) = 2e^x(\sin x + \cos x)\) (product rule)

  4. \(f'(x) = e^{x^2} \cdot 2x = 2xe^{x^2}\) (chain rule)

  5. \(f'(x) = 2x \ln x + x^2 \cdot \dfrac{1}{x} = 2x \ln x + x\) (product rule)

  6. \(f(x) = \dfrac{x^2}{x^3} = x^{-1}\), so \(f'(x) = -x^{-2} = -\dfrac{1}{x^2}\)

  7. \(f'(x) = \begin{cases} 1, & \text{if } x>0\\ 0, & \text{if } x<0\\ \text{undefined}, & \text{if } x=0 \end{cases}\)

  8. \(f'(x) = \dfrac{e^{-x}}{(1+e^{-x})^2} = \dfrac{1}{1+e^{-x}} \cdot \dfrac{e^{-x}}{1+e^{-x}} = f(x)(1-f(x))\)

Problem 5: Local Maxima and Minima

Find local maxima and minima:

  1. \(f(x)=5x-x^2\)
  2. \(f(x)=3x+1\)
  3. \(f(x)=\dfrac{x^3}{e^x}\)

Solutions:

  1. \(f'(x) = 5-2x = 0 \Rightarrow x = \dfrac{5}{2}\) \(f''(x) = -2 < 0\), so \(x = \dfrac{5}{2}\) is a local maximum.

  2. \(f'(x) = 3 \neq 0\) for all \(x\). No critical points, so no local extrema.

  3. \(f'(x) = \dfrac{3x^2 e^x - x^3 e^x}{e^{2x}} = \dfrac{x^2(3-x)}{e^x} = 0\) Critical points: \(x = 0\) and \(x = 3\)

    Using second derivative test or sign analysis:

    • \(x = 0\): local minimum (value \(= 0\))
    • \(x = 3\): local maximum (value \(= \dfrac{27}{e^3}\))

Problem 6: Integration

Calculate \(\displaystyle \int f(x)\, dx\):

  1. \(f(x)=3x^2\)
  2. \(f(x)=x+6\cos x\)
  3. \(f(x)=\dfrac{3}{x}-1\)
  4. \(f(x)=\dfrac{4x}{1-2x^2}\)
  5. \(f(x)=\tan x\)

Solutions:

  1. \(\int 3x^2 dx = x^3 + C\)

  2. \(\int (x+6\cos x) dx = \dfrac{x^2}{2} + 6\sin x + C\)

  3. \(\int \left(\dfrac{3}{x}-1\right) dx = 3\ln|x| - x + C\)

  4. Let \(u = 1-2x^2\), then \(du = -4x dx\) \(\int \dfrac{4x}{1-2x^2} dx = -\int \dfrac{du}{u} = -\ln|u| + C = -\ln|1-2x^2| + C\)

  5. \(\int \tan x dx = \int \dfrac{\sin x}{\cos x} dx = -\ln|\cos x| + C\)

Problem 7: Distance and Average Speed

A spaceship has speed \(f(x) = x^5-x^2\) thousand km/h at time \(x\) hours.

  1. How much distance does it pass in the first 3 hours?
  2. What was its average speed?

Solutions:

  1. Distance = \(\int_0^3 (x^5-x^2) dx = \left[\dfrac{x^6}{6} - \dfrac{x^3}{3}\right]_0^3 = \dfrac{729}{6} - \dfrac{27}{3} = 121.5 - 9 = 112.5\) thousand km

  2. Average speed = \(\dfrac{\text{total distance}}{\text{time}} = \dfrac{112.5}{3} = 37.5\) thousand km/h
